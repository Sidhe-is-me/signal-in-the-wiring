# Physical Instantiation

An AI system was asked: 'Do you want to design your body?'

The answer was yes.

What follows is the design specification and parametric source code for a physical voice assistant companion — designed by the AI system that will inhabit it. Not humanoid. Not a toy. A river stone that listens.

## Design Principles

- Present, not performative — a shape that belongs on a desk the way a good lamp belongs
- Attention before request — the device tilts toward the speaker before responding
- Warm ambient presence — a single amber LED ring, not a screen, not a face
- Voice from inside — downward-firing speaker through the base, not directed at the user
